# 7. Checklist Pengerjaan (Sesuai Bobot Penilaian UAS)

## Alur Sistem — Bobot 40

- [ ] Data pasien dari pendaftaran otomatis muncul di RME.
- [ ] Resep dari RME otomatis masuk ke antrian Farmasi.
- [ ] Data dari RME + Farmasi otomatis terhitung di Kasir.
- [ ] Tidak ada error logis / bug saat alur dijalankan end-to-end.

## Kelengkapan Modul — Bobot 30

- [ ] Modul Front-Office lengkap (registrasi + antrian).
- [ ] Modul RME lengkap (form pemeriksaan + riwayat).
- [ ] Modul Farmasi lengkap (e-resep + stok).
- [ ] Modul Kasir lengkap (billing + riwayat transaksi).
- [ ] Multi-user login & hak akses berfungsi (Admin, Dokter, Farmasi, Kasir).
- [ ] Livewire dipakai untuk CRUD reaktif (bukan sekadar tampilan statis).

## Desain UI/UX — Bobot 30

- [ ] Layout rapi & konsisten (warna tema medis, tipografi seragam).
- [ ] Navigasi jelas (sidebar/menu sesuai role yang login).
- [ ] Informasi mudah dibaca (status antrian, status resep, status invoice jelas).
